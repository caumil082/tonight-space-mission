# 🌌 오늘밤 우주미션 (Tonight) — 기획서

> **오늘밤 우주미션** (영문명 **Tonight**) = "오늘 밤, 하늘에서 무슨 일이?"
> 공공데이터를 단순히 보여주는 게 아니라, **학생이 직접 탐구하는 활동**으로 바꿔주는 천문 앱.
>
> 작성일: 2026-06-11
> 버전: **v1.0 — 첫 버전 완료** ✅ (자세한 기능 목록은 [CHANGELOG.md](CHANGELOG.md))
> v1 핵심 기능(오늘·달력·미션·일지·퀴즈 + 추천 히어로 + 백업)이 모두 구현·점검되었습니다.

---

## 1. 한 줄 소개

> **중학생이 "오늘 밤 볼 수 있는 천문현상"을 확인하고, 미션과 퀴즈로 직접 탐구하는 게임형 천문 앱**

단순 정보 조회가 아니라 → **게임형 천문 달력 + 관측 일지 + 퀴즈 앱**

---

## 2. 왜 만드나요? (해결하려는 문제)

- **지금의 불편함**: 공공데이터포털의 천문현상 API는 **글자 위주**라서, 중학생이 보기엔 딱딱하고 지루합니다. "동방최대이각" 같은 용어도 어렵습니다.
- **이 앱이 있으면**:
  - 어려운 용어를 **쉬운 말**로 풀어주고,
  - "오늘 밤 볼 만한 것"을 **카드처럼** 한눈에 보여주고,
  - **미션과 퀴즈**로 직접 참여하게 만들어,
  - 단순 조회가 아니라 **나만의 과학 탐구 기록**이 됩니다.

> 💡 핵심 아이디어: "오늘 뭐가 있지?" → "그게 무슨 뜻이지?" → "내가 직접 봐야지!" 로 자연스럽게 이어지게 만들기.

---

## 3. 누가 쓰나요? (사용자)

- **주 사용자: 중학생** (천문에 관심 있는 학생, 과학 수행평가/탐구활동)
- 로그인·회원가입 **불필요** → 만들기도 쉽고, 쓰기도 간편
- 인터넷 연결 없이도 사용 (오프라인 동작) — 아래 7번 "데이터 방식" 참고

> 💡 혼자 쓰는 앱이라 "계정", "공유", "보안 로그인" 같은 복잡한 기능을 처음부터 만들 필요가 없어요. 초보자에게 아주 유리한 조건입니다.

## 4. 핵심 기능 (MVP — 최소 기능 제품)

> v1(첫 버전)에 **반드시 넣을 핵심 기능**과, 함께 들어가는 게임 요소(미션·퀴즈)입니다.
> 미션을 수행하면 그 결과가 **관측 일지 한 줄로 자동 기록**되도록 연결했습니다.

| 우선순위 | 기능 | 설명 | 중학생 흥미 요소 |
|------|------|------|------------------|
| 1️⃣ | **오늘의 천문현상 카드** | 오늘 + 다가오는 며칠의 현상을 카드로 표시 | "오늘 밤 볼 만한 것" 즉시 확인 |
| 2️⃣ | **이번 달 우주 캘린더** | 한 달 현상을 달력으로 + 오늘/이번주/이번달 보기 | 유성우·보름·행성 접근을 한눈에 |
| 3️⃣ | **쉬운 설명 카드** | 합삭·망·최대이각 등을 쉬운 말로 변환 | 어려운 용어 장벽 낮춤 |
| 4️⃣ | **관심 이벤트 저장 (☆)** | 보고 싶은 현상을 ☆로 저장 | "나중에 다시 볼 이유" 생김 |
| 5️⃣ | **관측 일지** | 결과·하늘상태·장소·느낀점 기록 | 탐구 포트폴리오 |
| ＋ | **우주 미션** | 현상별 미션 → 결과가 일지로 기록 (시도도 점수!) | 게임처럼 참여 |
| ＋ | **퀴즈 모드** | 오늘/이번 달 현상 기반 3문제 | 학습 효과 강화 |

> 💡 **3️⃣ 쉬운 설명**은 별도 화면이 아니라, 모든 현상 카드 안에 💡로 함께 표시됩니다.
> **4️⃣ 관심(☆)**도 별도 탭 없이 카드의 ☆ 버튼으로 등록하고, '오늘' 탭 맨 위에 모아 보여줍니다.

### 각 기능 좀 더 풀어보기

**① 오늘의 우주 이벤트**
- API는 "월별"로만 주기 때문에, **한 달 전체를 미리 받아두고** 앱 안에서 `오늘 / 이번 주 / 이번 달`로 다시 나눕니다.
- 🌟 **대표 이벤트 히어로 카드**: 앱을 열면 맨 위에 "가장 볼만한 현상"을 큰 카드로 보여줍니다(D-day·쉬운설명·미션 바로가기). → "오늘 진짜 볼 게 있네!"를 즉시 느끼게.
- ⭐ **추천 배지**: 현상마다 '볼만함 점수'를 매겨 `🌟강력추천`/`⭐추천`을 표시. 유성우(ZHR 큰 것)·행성 충·보름달·식 등이 높은 점수, "국내 관측 불가"는 제외.
- 각 현상 카드에 표시: 날짜, 현상 이름, 시간, **관측 난이도**(🟢보기 쉬움 / 🟡보통 / 🔴어려움), D-day.
- 난이도는 1차 버전에서는 **간단한 규칙**으로 정합니다.
  - 🟢 보기 쉬움: 보름달, 반달 등 (맨눈)
  - 🟡 보통: 행성 접근, 최대이각 (쌍안경 있으면 좋음)
  - 🔴 어려움: 망원경 필요, 새벽 시간대 현상
  - (날씨·미세먼지·지역별 일몰은 1차 버전에서는 다루지 않음 → 나중 확장)

**② 쉬운 설명 (용어 사전)**
- 앱 안에 작은 **용어 사전**을 둡니다. 예:
  | 원문 용어 | 쉬운 설명 |
  |-----------|-----------|
  | 합삭 | 달이 태양 방향에 있어 거의 보이지 않는 날 |
  | 망 / 보름 | 달이 둥글게 보이는 시기 |
  | 최대이각 | 행성이 태양에서 가장 멀리 떨어져 보이는, 관측하기 좋은 때 |
  | 근지점 | 달이 지구에 가장 가까워져 조금 커 보이는 때 |
- 현상 카드를 누르면 그 안의 어려운 단어를 자동으로 풀어 설명합니다.

**③ 우주 미션** ⭐
- 현상 종류에 맞는 미션을 자동으로 제시합니다.
  | 미션 | 언제 나오나 |
  |------|-------------|
  | 🌕 보름달 스케치 미션 | 보름/망 현상일 때 |
  | 🪐 행성 찾기 미션 | 행성 접근·최대이각일 때 |
  | ☄️ 별똥별 세기 미션 | 유성우일 때 |
  | 📣 가족에게 1분 설명 미션 | 그 외 현상 |
- **핵심 원칙: 성공만 점수 주지 않습니다.** 날씨 때문에 못 봐도 "관측 시도"를 기록하면 점수를 줍니다.
  - 성공 30점 / 일부 관측 20점 / 시도(실패) 10점
- 점수가 모이면 **배지**가 올라갑니다. (🌱 입문 → ✨ 새내기 → 🌙 달 관찰자 → 🔭 별 사냥꾼 → 🚀 우주 탐험가)

**④ 퀴즈 모드** ⭐
- 오늘(또는 이번 달) 현상 + 용어 사전을 재료로 **3문제**를 자동으로 만듭니다.
- 문제 유형 예시:
  - "오늘 밤 볼 수 있는 천문현상은?" (보기 4개 중 고르기)
  - "'최대이각'은 무슨 뜻일까요?" (용어 뜻 맞히기)
  - "이 현상의 관측 난이도는?" (🟢/🟡/🔴)
- 풀고 나면 정답·해설을 보여주고 점수를 기록합니다.

**＋ 이번 달 우주 캘린더**
- 달력 격자에 현상이 있는 날을 **점(개수)** 으로 표시, 오늘은 강조.
- 날짜를 누르면 그날 현상이 아래에 카드로 나옵니다.
- 빠른 보기 칩: **오늘/선택일 · 이번 주 · 이번 달 전체**, 그리고 ◀▶ 로 달 이동.

**＋ 관심 이벤트 (☆)**
- 카드의 ☆를 누르면 저장(★), '오늘' 탭 맨 위 `⭐ 내 관심 이벤트`에 모임.
- 저장은 브라우저 안(localStorage)에 — 회원가입 불필요.

**＋ 관측 일지 (미션과 연결!)**
- 입력 항목: 관측 결과(성공/일부/시도) · 하늘 상태 · 장소 · 느낀 점 · **사진**.
- **미션 → 일지 연결**: 미션에서 결과 버튼을 누르면 일지에 자동으로 한 줄이 생기고,
  그 자리에서 하늘·장소·느낀 점·사진을 덧붙일 수 있습니다.
- **직접 기록 추가**: 미션과 상관없는 관측도 남길 수 있고, **지난 날짜(과거 관측)** 로도 기록 가능.
- **사진 첨부**: 업로드하면 자동으로 작게 압축해 저장(브라우저 안에만 보관 → 개인정보 안전).
- 점수·배지는 이 일지 기록들을 기준으로 계산됩니다.

## 5. 화면 구성 (대략적인 그림)

아래 **5개 탭**으로 시작합니다. (쉬운 설명·관심☆은 카드 안에 녹아 있어요.)

```
┌───────────────────────────────────────────┐
│  🌌 오늘밤 우주미션     🌱 관측 입문 · 0점  │ ← 상단: 앱 이름 + 내 점수/배지
├───────────────────────────────────────────┤
│  [오늘] [달력] [미션] [일지] [퀴즈]         │ ← 탭 5개
├───────────────────────────────────────────┤
│  오늘 6월 11일 (목)                         │
│  ┌─────────────────────────────────────┐  │
│  │ 🪐 달과 토성 근접          오늘 03:00 │  │ ← 현상 카드
│  │ 🔴 어려움 · 새벽 동쪽 하늘            │  │
│  │ "근접 = 두 천체가 가까이 보이는 때"   │  │ ← 쉬운 설명
│  └─────────────────────────────────────┘  │
│  ┌─────────────────────────────────────┐  │
│  │ 🌕 보름달(망)              D-18       │  │
│  │ 🟢 보기 쉬움                          │  │
│  └─────────────────────────────────────┘  │
└───────────────────────────────────────────┘
```

- **[달력] 탭**: 달력 격자(현상 있는 날에 점) + 오늘/이번주/이번달 보기
- **[미션] 탭**: 현상별 미션 카드 → "성공 / 일부 / 시도" → 일지로 자동 기록
- **[일지] 탭**: 미션 기록 + 직접 기록 모음 (결과·하늘·장소·느낀점)
- **[퀴즈] 탭**: 3문제 풀기 → 점수와 해설

> 밤하늘 느낌의 **어두운(다크) 배경 + 별** 디자인으로 중학생 취향에 맞춥니다.

## 6. 만드는 순서 (단계별 계획)

> 초보자에게 가장 중요한 건 **작게 시작해서 하나씩 늘리는 것**입니다.

### 🟢 1단계: 데이터 준비
- 공공데이터포털 천문현상 API에서 **1년치를 한 번만** 받아 JSON 파일로 저장
- 앱은 그 파일만 읽음 → 인터넷·서버·추가 비용 없이 동작

### 🟢 2단계: 화면 만들기 (브라우저에서 동작)
- "오늘의 우주 이벤트" 카드 + 쉬운 설명 표시
- 데이터는 앱 안에 들어있으므로 바로 보임

### 🟢 3단계: v1 핵심 기능 채우기 ✅ (완료)
- 🗓 **이번 달 캘린더**: 달력 + 오늘/이번주/이번달 보기
- ☆ **관심 이벤트 저장**, 📒 **관측 일지**(미션 결과와 연결, 과거 날짜·사진)
- 🎯 **우주 미션** + 📝 **퀴즈**: 자동 생성 + 채점/점수/배지
- 🌟 **대표 이벤트 추천**(히어로 카드), 💾 **백업/복원**(사진 포함)

### 🔵 4단계 이후: 천천히 추가하면 좋은 것들
- 🌤 날씨/미세먼지 연동으로 "오늘 밤 진짜 볼 수 있나" 정확도 높이기
- 📱 Tauri로 Windows 앱(.exe) 만들기 (StepDay처럼)
- ☁️ (선택) 여러 기기 자동 동기화 — 지금은 파일 백업으로 대체

## 7. 기술 방향 (어떻게 만들지)

> StepDay와 같은 방식이라 초보자에게 익숙하고 안전합니다.

| 항목 | 선택 | 이유 |
|------|------|------|
| **앱 형태** | 웹앱(HTML/CSS/JS) 먼저 → 나중에 Tauri | 배우기 쉽고 자료 많음. 빌드 도구 없이 시작 |
| **데이터 저장** | 점수·기록은 브라우저 안(로컬)에 저장 | 서버·로그인 불필요, 오프라인 동작 |
| **천문 데이터** | **1년치를 미리 받아 앱에 번들** | 아래 설명 참고 |

### ⚠️ 가장 중요한 결정: 천문 데이터를 "미리 받아두기"

공공데이터 API를 **브라우저에서 직접** 부르면 문제가 생깁니다:
- `apis.data.go.kr`은 보통 **CORS 차단** → 브라우저에서 부르면 실패
- **서비스키가 그대로 노출**됨
- 매번 호출하면 **오프라인 불가** + 호출량 늘어남

천문현상은 **1년치가 작고, 미래 날짜까지 거의 정해져 있어서**, 다음 방식이 가장 좋습니다:

```
① 딱 한 번:  node 스크립트로 1년치(월 12번) 호출 → JSON 1개로 저장
② 그 다음:   앱은 그 JSON 파일만 읽음 (호출 0번, 비용 0원, 오프라인 OK)
③ 1년 뒤:    스크립트를 한 번 더 돌려 데이터 갱신
```

> 이렇게 하면 **"학생의 데이터 사용 비용 최소화"** 라는 목표가 그대로 달성됩니다.
