## v1.0 — 2026-06-11 (첫 버전 🎉)

중학생용 게임형 천문 앱 **"오늘밤 우주미션"(Tonight)** 의 첫 버전.
공공데이터를 단순 조회가 아니라 **미션·퀴즈·기록**으로 바꿔 직접 탐구하게 만듭니다.

### 기능 (탭 5개)
- **오늘** — 오늘 + 다가오는 30일 현상 카드 (관측 난이도 · D-day · 쉬운 설명)
- 🌟 **대표 이벤트 히어로 카드** — "볼만함 점수"로 가장 좋은 현상을 첫 화면에 크게 (친근한 제목 · 후킹 카피 · 관측 꿀팁)
- ⭐ **추천 배지** — 현상마다 강력추천/추천 표시
- **달력** — 이번 달 캘린더(현상 있는 날에 점) + 오늘/이번주/이번달 보기
- **미션** — 현상별 미션. 결과(성공/일부/시도)가 **관측 일지로 자동 기록** ("시도"도 점수!)
- **일지** — 미션 기록 + 직접 기록. **과거 날짜 입력 · 사진 첨부**(자동 압축)
- **퀴즈** — 오늘/이번 달 현상 기반 3문제
- ☆ **관심 이벤트 저장**
- 💾 **백업/복원** — 사진 포함 전체 기록을 JSON 파일로 내보내기/불러오기

### 데이터
- 한국천문연구원 천문현상 정보 API에서 **2026년 186건**을 미리 받아 번들 → 오프라인 동작, 추가 비용 0
- 갱신: `node tools/fetch-astro.mjs --key=Decoding키 --year=2026`
- (2027년은 아직 API 미공개 — 연말쯤 재수집 예정)

### 기술
- 빌드 도구 없는 순수 HTML/CSS/JS, 점수·기록은 브라우저 localStorage에 저장
- 스크립트는 `?v=숫자`로 불러와 캐시 문제 방지 (코드 수정 시 숫자 올리기)

### 점검 (QA)
- 여러 날짜(연초·연말·데이터 이후 2027) · 빈 상태 · 백업 라운드트립 · 캐시 · 핵심 플로우 전수 점검 통과 (콘솔 에러 0)
