# 🌌 오늘밤 우주미션 (Tonight)

중학생을 위한 **게임형 천문 앱**. 공공데이터(한국천문연구원 천문현상 정보)를 단순 조회가 아니라
**미션 · 퀴즈 · 쉬운 설명**으로 바꿔, 직접 탐구하게 만듭니다.

> 기획 의도와 전체 그림은 [`기획서_초안.md`](기획서_초안.md) 를 보세요.

---

## 빠르게 실행하기

가장 쉬운 방법: **`index.html` 을 더블클릭**해서 브라우저로 열면 끝!
(데이터가 앱 안에 들어 있어서 인터넷 없이도 바로 동작합니다.)

---

## 폴더 구조

```
Tonight/
├─ index.html            ← 앱 시작점 (이걸 열면 됨)
├─ css/style.css         ← 디자인 (밤하늘 다크 테마)
├─ js/
│  ├─ data.js            ← 날짜·용어사전·난이도 도우미
│  ├─ favorites.js       ← 관심 이벤트 저장 (☆)
│  ├─ journal.js         ← 관측 일지 + 점수/배지
│  ├─ backup.js          ← 백업/복원 (사진 포함, JSON 파일)
│  ├─ recommend.js       ← 추천 점수 + 대표 이벤트 선정
│  ├─ missions.js        ← 우주 미션 만들기
│  ├─ quiz.js            ← 3문제 퀴즈 만들기
│  └─ app.js             ← 화면 그리기 (본체)
├─ data/events.js        ← 천문현상 데이터 (지금은 예시 데이터)
├─ tools/fetch-astro.mjs ← 실제 데이터를 받아오는 스크립트
└─ 기획서_초안.md
```

## 실제 데이터로 바꾸기 (한 번만 하면 됨)

지금 앱에는 동작 확인용 **예시 데이터**가 들어 있어요.
공공데이터포털 서비스키가 있으면 진짜 데이터로 교체할 수 있습니다.

```bash
node tools/fetch-astro.mjs --key=여기에_서비스키 --year=2026
```

- `--key`  : 공공데이터포털에서 받은 **일반 인증키(Decoding)** 권장
- `--year` : 받을 연도 (기본값 2026)

성공하면 `data/events.js` 가 실제 데이터로 자동 교체되고,
앱 상단의 "⚠️ 예시 데이터" 안내가 "✅ 실제 데이터"로 바뀝니다.

> 천문현상은 미래까지 거의 정해져 있어서, **1년에 한 번만** 받아두면 충분합니다.
> 그래서 평소엔 API를 부르지 않아 **데이터 사용 비용이 0원**입니다.

## 시연 팁

오늘 날짜에 현상이 없어 화면이 비어 보이면, 주소 끝에 날짜를 붙여
"그 날을 오늘처럼" 볼 수 있어요. 예:

```
index.html?today=2026-06-11
```

## v1에 들어간 기능 (탭 5개)

- **오늘** — 🌟 대표 이벤트 히어로 카드(가장 볼만한 현상 + D-day) + 오늘/다가오는 30일 현상 카드(추천 배지·난이도·D-day·쉬운 설명), 관심☆ 모음
- **달력** — 이번 달 캘린더(현상 있는 날에 점) + 오늘/이번주/이번달 보기
- **미션** — 현상별 미션. 결과(성공/일부/시도)를 누르면 **관측 일지로 자동 기록** *("시도"도 점수!)*
- **일지** — 미션 기록 + 직접 기록 모음 (결과·하늘상태·장소·느낀점). **과거 날짜 입력**과 **사진 첨부**(자동 압축) 지원 → 탐구 포트폴리오. 맨 아래에서 **백업 파일 내보내기/불러오기**(사진 포함)
- **퀴즈** — 오늘/이번 달 현상으로 만든 3문제

> 쉬운 설명과 관심☆은 별도 탭 없이 카드 안에 들어 있어요.

다음에 추가하면 좋은 것: 날씨 연동, 2027년 데이터, 미션 종류 추가, Tauri Windows 앱.
